# Patch_Yolo

Adversarial Patch knowledge dump (YOLO)

## Overview

This repository serves as a knowledge base for adversarial patch attacks targeting YOLO (You Only Look Once) object detection models. It collects research notes, experiments, references, and implementation details related to physically realizable adversarial patches.

## Contents

- Research notes and paper summaries on adversarial patches
- YOLO model attack experiments and findings
- Implementation references and code snippets

## Background

Adversarial patches are perturbations, typically printed stickers or overlays, that can fool deep-learning-based object detectors like YOLO when placed in a scene. This project documents techniques, results, and insights gathered during exploration of this attack surface.

## Getting Started

Browse the files in this repository to explore the collected knowledge. New notes and findings will be added over time.
